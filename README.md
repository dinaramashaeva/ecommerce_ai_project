# AI-Integrated E-Commerce Web Application

This project is a full-stack AI-powered e-commerce web application developed as part of the MAT 900 Final State Certification Project.

## Technologies Used

Frontend:
- React.js
- Tailwind CSS

Backend:
- Node.js
- Express.js

Database:
- PostgreSQL

AI Integration:
- Google Gemini API

## Features

- User registration and authentication (JWT)
- Product listing and product details
- AI-powered product search
- Shopping cart functionality
- Order management system
- Admin dashboard for product control
- REST API architecture

## Project Structure

/client → React frontend  
/server → Node.js backend  

## Installation

Clone repository:
git clone https://github.com/dinaramashaeva/ecommerce_ai_project

Install dependencies:


cd server
npm install

cd ../client
npm install

Run backend:


npm run dev


Run frontend:


npm run dev
